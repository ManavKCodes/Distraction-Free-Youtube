# 🎬 Distraction-Free YouTube

A **YouTube-inspired web application** built entirely with **HTML** that provides a clean and distraction-free browsing experience. Instead of endless recommendations and clutter, users can directly choose a category and access curated educational and entertainment videos.

---

## 📌 About the Project

Distraction-Free YouTube is a simple frontend project designed to reduce unnecessary distractions while watching videos. The website organizes content into different categories, allowing users to quickly find videos related to their interests.

This project was built as a **real-world HTML project** to strengthen my understanding of webpage structure, navigation, hyperlinks, semantic HTML, and multi-page website development.

---

## ✨ Features

* 🎯 Clean and distraction-free interface
* 📂 Multiple category pages
* 💪 Fitness videos
* 💻 Programming tutorials
* 🎮 Gaming videos
* 🎵 Music section
* 😂 Comedy section
* 🏠 Home page with category navigation
* 🔙 Back-to-Home navigation on every page
* 📱 Beginner-friendly project structure

---

## 🛠️ Tech Stack

* HTML5

> No CSS frameworks or JavaScript libraries were used.
